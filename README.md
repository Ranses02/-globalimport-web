# Comercio Electrónico - GlobalImport

Este proyecto es un sitio web profesional de comercio electrónico para una empresa importadora de productos.

## Objetivo
Crear una presencia web clara y moderna para GlobalImport, con una estructura escalable que incluya:
- Página de inicio con propuesta de valor.
- Catálogo de productos destacados.
- Sección de beneficios.
- Área de contacto.

## Etapas del proyecto
1. Planificación: definir secciones, estilo y contenidos.
2. Maquetación HTML: construir la estructura semántica del sitio.
3. Estilos CSS: aplicar diseño responsive y presentación visual.
4. Contenido: agregar textos y ejemplos de productos.
5. Interacción: implementar navegación móvil y carrito básico si es necesario.
6. Revisión: probar en distintos dispositivos y ajustar detalles.
